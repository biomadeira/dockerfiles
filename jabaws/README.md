# JABAWS

[JABAWS](http://www.compbio.dundee.ac.uk/jabaws22/) Distributed Web Services for Bioinformatics.

## Example Usage

Pull the image:
```
docker pull bartongroup/jabaws:2.2
```

To start a container:
```
docker run --rm -it -p 8080:8080 bartongroup/jabaws:2.2
```
